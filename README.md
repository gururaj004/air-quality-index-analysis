# Air Quality Index Trend & Pollution Source Diagnosis

## Project Overview
This project focuses on analyzing Air Quality Index (AQI) patterns across different monitoring stations and time periods in India using Exploratory Data Analysis (EDA).

The analysis aims to identify:
- Major pollution contributors
- Seasonal AQI variations
- High-risk regions
- Pollutant trends over time

---

## Problem Statement
A municipal government's environmental department needs to understand air quality patterns across monitoring stations and time periods.

Citizens are reporting increased respiratory issues, and the department wants to identify:
- Which pollutants are most harmful
- Which seasons are worst
- Which areas need urgent policy intervention

---

## Dataset Information
Dataset: Air Quality Data in India (2015–2020)

Features include:
- PM2.5
- PM10
- NO2
- SO2
- CO
- O3
- AQI values
- AQI category labels
